How this stays alive on Render (free tier)
The server pings itself at /ping every 14 minutes — no external cron service needed. Render's free tier sleeps after 15 min of inactivity; this prevents that.

Deploy steps:
1. Push this folder to a GitHub repo
2. New Web Service on Render → connect the repo
3. Build: npm install   Start: npm start
4. Set env var RENDER_URL = your .onrender.com URL
5. Save your weather key in this Settings page — done ✓
